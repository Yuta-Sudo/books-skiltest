プログラミングスキルテスト
問題 下記要件を満た書籍おすすめサービスをPHP・MySQLをベースに作成してください

機能要件

必須
	新規会員登録機能
		パスワードは4文字以上16文字以内のバリデーションチェックが必要
		その他項目は必須入力のバリデーションチェックが必要
		画像アップロード機能

	ログイン・ログアウト機能

	おすすめ書籍一覧表示機能
	おすすめ書籍詳細表示機能

	おすすめ書籍アップ機能
		本のタイトル
		おすすめ理由
		本の写真


	おすすめ書籍編集機能

	おすすめ書籍削除機能

	プロフィール編集機能

拡張機能(時間があれば挑戦してください)
	おすすめ書籍タイトル検索機能
	いいね機能
	ページング機能
	その他なんでも

システム要件
	PHP version 5.5以上
	MySQL version 5.6以上

提出物
	テーブル定義書(MySQL上で動作するCreate Table文も付記すること)
	プログラムフォルダを載せたGithub URL

提出方法
	GoogleDriveの​提出用フォルダ​にテーブル定義書を提出
	テーブル定義書テンプレートは下記を自由にお使いください
	上記を終えた時点で期限時間内に提出完了の旨を下記メール提出先アドレス宛に送信 ○ Github URLはこのメールに必ず記載
	提出メールの件名は下記フォーマットを使用

メール提出先アドレス
スキルテスト提出完了(yyyy_mm_dd Firstname_Lastname)
To
	mfujimoto@nexseed.net
CC
	eichinohe@nexseed.net

評価ポイント
1. 機能要件通りの実装

2. 資料全体の正確性・メールや提出方法等指示どおりの対応

3. UXを考慮したデザイン
○ 1項の実装部分が最重要事項ですが、デザインも最低限現代WEBサービスを考慮できているか
○ 凝ったデザインやJSなどを使用したエフェクトはプラスαの評価

4. 拡張機能の実装
○ テーブル定義書に加味されているか、機能が実装されているか
